Quick Tips

Why is out-of-date software so dangerous?

If you spend any amount of time on a computer, then you likely use at least a few different software that regularly need to be updated. This could look like an update to your Microsoft 365, Spotify, or Adobe software. Often these updates can be frustrating as they cause us to stop our normal workflow and wait for the update to complete.

Once a new update is released older versions of that software can become targets for attack. The newer version likely includes security updates, while the older versions could have vulnerabilities that hackers learn about over time. By using an older version of a software, you’re opening the door for attackers.


Advice

Set aside a short period of time each to week to complete any available updates on your computer.
